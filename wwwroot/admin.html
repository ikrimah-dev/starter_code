Qurra Central Admin

Use this page to create masjid organisers, add new events, and attach comments to existing events. The forms will later send POST requests to the API.

Create New Masjid Organiser

Add a new masjid or organising body that will be linked to events.

Create New Event

Link the event to a masjid organiser and specify the type of programme.

Add Comment to Event

Attach attendee feedback to an existing event using its event ID.
